- **Frontend**: React (Vite), Framer Motion, React Icons.

- **Data Engineering**: PostgreSQL, MongoDB, SQL, NoSQL.
- **Infrastructure**: Docker, Kubernetes, AWS EKS, CI/CD Pipelines.
- **Specialized**: System Design, Microservices, AI/LLM Orchestration (Groq/Llama-3), Unit Testing.

## 🛠 Features

- **Dynamic Animations**: Seamless, high-performance UI using Framer Motion.
- **Glassmorphic UI**: Premium, dark-mode-first aesthetic with curated color palettes.
- **AI Integration**: Custom logic for data extraction and LLM orchestration.
- **Contact Redirect**: Direct link to Gmail for seamless client communication.

## 🚀 Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run Development Server**:
   ```bash
   npm run dev
   ```

3. **Build for Production**:
   ```bash
   npm run build
   ```
